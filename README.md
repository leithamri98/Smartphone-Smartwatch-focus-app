# Smartphone-Smartwatch Focus App

A Kotlin-based mobile application designed to help workers stay focused, boost creativity, and achieve their goals through intelligent task management and focus level tracking across smartphone and smartwatch devices.

## Project Overview

This application assists workers in maintaining concentration and improving productivity by providing a flexible scheduling system with customizable focus levels. The app is designed to run seamlessly on both smartphones and smartwatches, offering a unified experience across devices.

## Purpose

The goal of this project is to:
- Help workers maintain focus and concentration during work tasks
- Enhance creativity and productivity through structured work sessions
- Promote mental well-being by encouraging healthy work habits
- Enable users to track their progress and personal development
- Reduce time wastage and improve goal achievement

## Key Features

### 1. Task Management
- Create and manage multiple tasks with flexible scheduling
- Assign custom timers for each task
- Organize tasks based on priority and deadline

### 2. Focus Levels
The app offers **3 customizable focus levels** to suit different work scenarios:
- **Low Focus**: For casual tasks and light work
- **Intermediate Focus**: For regular work tasks requiring moderate concentration
- **High Focus**: For critical tasks demanding maximum concentration and minimal distractions

### 3. Timer System
- Individual task timers with visual and audio feedback
- Real-time progress tracking
- Adjustable time allocations per task

### 4. Self-Revision & Reflection
- Intelligent agent-based questioning system
- Post-task behavioral assessment
- Users reflect on their performance and focus during the task
- Helps identify productivity patterns and areas for improvement

### 5. Cross-Device Synchronization
- Seamless sync between smartphone and smartwatch
- Start/pause/resume tasks from either device
- Real-time notifications and updates
- Optimized user interface for each device

### 6. Progress Tracking
- Visual dashboard showing productivity metrics
- Task completion history
- Focus level performance analysis
- Personal growth insights

## Technology Stack

- **Language**: Kotlin
- **Platforms**: 
  - Android (Smartphone)
  - Wear OS (Smartwatch)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Database / Firestore (for cloud sync)

## Main Activities

The app consists of **3 main activities**:

1. **Task Scheduler Activity**
   - Create, edit, and manage tasks
   - Set task duration and focus levels
   - View all scheduled tasks

2. **Focus Session Activity**
   - Start focused work sessions
   - Timer display with focus level indicator
   - Real-time notifications
   - Pause/resume functionality

3. **Self-Revision & Reflection Activity**
   - Post-session questionnaire
   - Behavioral analysis questions
   - Progress review and insights
   - Performance metrics and statistics

## User Interface

- **Smartphone UI**: Full-featured interface for task management and detailed analytics
- **Smartwatch UI**: Simplified, glance-able interface for quick task access and timer monitoring

## Expected Outcomes

By using this application, users can expect to:
